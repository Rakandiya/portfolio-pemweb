RPP Guru

Web aplikasi ini dapat melakukan export data menjadi file pdf untuk setiap satu RPP. Web aplikasi ini hanya bisa diakses oleh guru dan karyawan di SMKN 1 Garut dan tidak bisa diakses oleh siswa. Web aplikasi ini dibuat menggunakan framework Codeigniter sebagai backend dan Bootstrap sebagai frontend.

GDS

GDS (Gerakan Disiplin Siswa) adalah sebuah aplikasi mobile yang digunakan untuk melakukan pencatatan pelanggaran siswa di SMKN 1 Garut. Didalamnya terdapat fitur untuk menghitung berapa kali siswa tersebut melakukan pelanggaran, jika sudah mencapai jumlah tertentu, maka siswa tersebut akan diberikan sanksi yang menggunakan konsep bintang, dimana setiap kali melakukan pelanggaran, poin bintang tersebut berkurang. Aplikasi ini dibuat menggunakan React Native sebagai mobile dan Laravel sebagai backend.
